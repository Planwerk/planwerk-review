# Reference

Information-oriented, exhaustive lookup. These pages describe the tool's
surface as precisely as possible — they are for confirming a flag, a field, or
a value, not for learning.

- [CLI](/reference/cli) — every subcommand, argument, and flag.
- [Configuration file](/reference/configuration) — the `.planwerk/config.yaml`
  schema and precedence rules.
- [Review patterns](/reference/review-patterns) — pattern format, sources, and
  remote-source syntax.
- [Output format](/reference/output-format) — severity levels, finding fields,
  and the machine-readable report block.
- [GitHub Action](/reference/github-action) — inputs and outputs of the
  composite action.
- [Environment variables & exit codes](/reference/environment-variables).
- [Project structure](/reference/project-structure) — repository layout,
  workflows, and dependencies.

For task-oriented instructions see the [How-to guides](/how-to/); for
background and rationale see the [Explanation](/explanation/) section.
